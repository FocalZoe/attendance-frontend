# Project - AI 影像識別自動點名系統

## 使用工具

- 使用 Antigravity IDE 作為開發工具
- 使用 Ameba 開發板傳送 Json 資料到 Supabase
- 使用 YOLOv8 進行物件辨識以達成點名功能
- 使用 Supabase 作為資料庫，負責儲存學生資料、點名結果與人臉特徵
- 使用 Github repository 作為版本控制與程式碼託管
- 使用 Vercel 進行前端網頁託管

## 功能需求

- 完成一套結合 AI 影像辨識、雲端部署與資料庫管理的自動點名系統
- 能夠透過教室影像自動辨識學生身分，並將點名結果上傳至雲端資料庫，減少人工點名時間，提高課堂管理效率
